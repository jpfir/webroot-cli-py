# webroot-cli-py
webroot-cli in Python.

Extract informations from webroot.


# Usage
```
./webroot-cli.py --help

./webroot-cli.py --client exo --active --last --noverbose
```


# Configuration file
```
[webroot]
id = XxX
secret = XxX
email = XxX@domain.ext
password = XxX
key_exo = XxXx-XxXx-XxXx-XxXx-XxXx
```
